# Dangerous Meeting

Official website for Dangerous Meeting, built with Next.js, React and TypeScript.

## Production

- Official domain: `www.dangerousmeeting.com`
- Hosting platform: Vercel
- DNS/domain provider: Cloudflare

## Social Links

- [YouTube - Dangerous Meeting Live](https://www.youtube.com/@DangerousMeetingLive)
- [Instagram - @dangerousmeeting_live](https://www.instagram.com/dangerousmeeting_live/)

## Local Development

```bash
npm install
npm run dev
```

The local development server runs at:

```text
http://localhost:3001
```

## Scripts

- `npm run dev`: starts the local development server on `http://localhost:3001`.
- `npm run build`: creates the production build.
- `npm run start`: runs the production server on `http://localhost:3001` after a build.
- `npm run lint`: checks the project with ESLint.

## Before Deploying

```bash
npm run lint
npm run build
```
